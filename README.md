## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
### Local Development Setup
```bash
Clone the repository:
git clone https://github.com/BillohMBarry/Web3-Quiz.git
```

```bash
cd Web3-Quiz
```

```bash
npm install
```
```bash
npm run dev
```
The application will start runnint at `http://localhost:3000`
